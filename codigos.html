Python: Contador de Palabras
Un script sencillo en Python que cuenta las palabras en un texto dado.
# Contador de palabras en Python
texto = "Hola, este es un ejemplo de texto."
palabras = texto.split()
print(f"Número de palabras: {len(palabras)}")
